PassVault
Smart Security for the Modern Web
Team: ColdBlooded

Project Overview

PassVault is a secure password manager web application designed to help users store, analyze, and manage passwords safely. It also includes phishing detection and password strength analysis to enhance overall cybersecurity.

Key Features
Secure Signup/Login (SHA-256 hashing + salting)
Password Strength Checker
Password Generator
Encrypted Password Storage
Weak Password Detection
Login Alert System
Phishing Website Detection

 DSA Concepts Used
Hashing → Secure authentication
Linked List → Managing dynamic user data
Trees → URL parsing for phishing detection
Strings/Arrays → Password analysis

How It Works
User signs up and logs in
Dashboard provides security tools
Passwords are stored securely
Alerts and phishing detection run in background
